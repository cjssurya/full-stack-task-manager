Tech Stack Used:

Frontend: React (CDN), Tailwind CSS, Axios.

Backend: Node.js, Express.js, Sequelize ORM.

Database: PostgreSQL (Hosted on Neon).

Auth: JWT (JSON Web Tokens) & Bcryptjs for password hashing.

Features Implemented:

User Authentication: Secure Signup and Login using JWT.

CRUD Operations: Create and view tasks.

Multi-tenancy: Users can only see their own tasks (Data isolation).

Cloud Deployment: Backend on Render, Frontend on GitHub Pages.

Setup Steps:

Clone the repository.

Run npm install in the root folder.

Create a .env file with your database credentials.

Run node server.js to start the backend.

Open index.html via Live Server.
